🎨 Bayt Al Fann

Bayt Al Fann is a modern online art gallery platform designed to showcase and sell artwork. The website provides a clean and elegant user experience for visitors while offering a secure admin panel to manage artworks and gallery content.

🌐 Live Demo

https://bayt-al-fann.vercel.app

✨ Features

- 🎨 Elegant and responsive art gallery
- 🖼️ Showcase artwork with high-quality images
- 📱 Fully responsive design
- 🔐 Secure Admin Dashboard
- ➕ Add, edit, and delete artworks
- 🗂️ Organize artworks into categories
- ⚡ Fast and optimized performance

🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- Vercel (Deployment)

🚀 Installation
